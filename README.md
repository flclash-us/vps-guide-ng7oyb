[English] Production-ready Clash config templates: 分流规则, proxy groups, DNS optimization.

## Features

- Multiple proxy group strategies (auto-select, failover, load-balance)
- Optimized DNS settings (DoH/DoT/DoQ)
- Complete分流rules covering mainstream websites
- Platform-specific configs (Windows/macOS/Android/iOS/OpenWrt)

## Quick Start

1. Choose template based on your use case
2. Edit proxy-providers, fill in your subscription URL
3. Import into Clash for Windows

## Template Overview

| Template | Use Case | Rule Count |
|---------|---------|---------|

| `advanced.yaml` | Advanced, full-featured | ~500 |
| `gaming.yaml` | Gaming, low latency | ~100 |
